## Usage

All code pertaining to the paper is in the "project folder" directory.
Old code to be cleaned up is under the "old" directory.
Code in the "pre-processing" directory serves to clean the data to arrive at a usable dataset for use in this paper.
Only the "project folder" directory will be published.


# "project folder"

1) Execute scripts in "data processing" directory to clean the data, find active/inactive health facilities, and calculate reporting rates.
1.1) Execute "HF_to_DS_agg_imputed_conf_rdt_and_allout_with_rep_weights_AVG_activeHFs.R" script
1.2) Execute "making_HF_active.R" script

2) Code in "analysis for figures" will run the analysis and create the figures in the main text and SI

3) The "for Ov5s" folder will do steps (1) and (2) for the Over-5 group
